# CarDekho + CarTrade Used Cars Scraper Promotion Notes

## Positioning

Collect structured public Indian used-car listings from CarDekho and CarTrade for price research, inventory comparisons, and vehicle-market analysis.

## Best small demo

- Source: `cardekho`
- City: `Mumbai`
- Model: `Honda City`
- Maximum results: `1`
- Price filters: empty
- Proxy: off

Show the input, one clean vehicle row, and JSON or CSV export. Mention that selecting `both` combines records from both marketplaces.

## LinkedIn draft

I published a CarDekho + CarTrade Used Cars Scraper on Apify for structured Indian vehicle-market research.

It collects public listing fields such as price, model year, kilometres driven, fuel type, transmission, body type, ownership count, location, image URL, and listing URL.

The safest first run is intentionally small: CarDekho, Mumbai, Honda City, one result, no price filters, and direct requests. Inspect the output, then add CarTrade, more cities, or price filters.

Useful for used-car price research, inventory comparisons, and automotive BI datasets.

## Reddit or Discord draft

I built an Apify Actor that turns public CarDekho and CarTrade listings into structured JSON/CSV data. It supports marketplace selection, Indian cities, car models, optional INR price filters, and result limits.

For a low-cost check, run CarDekho + Mumbai + Honda City + one result with proxy off. Feedback on the schema and useful vehicle-market fields is welcome.
